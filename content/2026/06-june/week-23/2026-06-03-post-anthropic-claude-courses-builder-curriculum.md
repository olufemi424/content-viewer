## Hook

## Why this matters
Most people learn AI tooling from random threads and scattered docs. Anthropic's official courses repo gives solo builders a cleaner path: API basics, prompt engineering, evaluations, and tool use in one place, with examples you can actually work through.

## Mechanism
Anthropic's public courses repository currently lists five official courses:
- Anthropic API fundamentals
- Prompt engineering interactive tutorial
- Real world prompting
- Prompt evaluations
- Tool use

The repo notes that many examples favor Claude 3 Haiku to keep follow-along costs down. Anthropic's Learning page separately positions these materials as official guides and tutorials for working with Claude.

## Proof/use case
Real use case: if you're building a support bot, internal assistant, or content workflow, this sequence is basically your implementation stack.
Start with API fundamentals so you stop guessing at model settings.
Use prompt engineering to structure output.
Use prompt evaluations so you can measure regressions.
Use the tool-use course once the workflow needs search, retrieval, or actions.
