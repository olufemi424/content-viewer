## Hook
You can now start a coding task in terminal and keep it moving from your phone.

## Why this matters
If you build solo, context switching kills momentum. Remote Control means your agent keeps working while you are away from your desk, so reviews, follow-ups, and small corrections happen in dead time instead of waiting for your next laptop session.

## Mechanism
Anthropic’s Claude Code docs now describe cross-surface workflows where you continue terminal sessions from phone/browser using Remote Control and hand off between terminal, desktop, and web. Practically, that turns one-off prompts into persistent runs you can monitor and steer asynchronously.

## Proof/use case
Use case: start a refactor + test pass from terminal before a meeting, then check progress and send one corrective instruction from mobile. You return to a branch that is already closer to merge instead of restarting context from scratch. This matches Anthropic’s documented “Work from anywhere” flow and the tracker signal showing high developer uptake.
